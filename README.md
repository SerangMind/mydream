# MyDream

중학생이 짧은 선택형 질문을 통해 자신의 성향을 가볍게 탐색해보는 정적 웹 게임입니다.

## 개요

- 단일 파일 `index.html`로 구성된 HTML/CSS/JavaScript 프로젝트
- 총 5개의 질문에 답하면 성향 점수를 계산
- 결과 화면에서 `주성향 + 보조성향` 조합 제공
- 추천 직업과 공부 방식 힌트를 함께 제시

## 게임 구조

### 성향 유형

- `analysis`: 분석가
- `empathy`: 공감가
- `planning`: 기획가
- `creative`: 창작가
- `execution`: 실행가

### 진행 흐름

1. 시작 화면에서 탐험 시작
2. 5개의 상황형 질문에 4지선다로 응답
3. 선택지별 점수를 누적
4. 점수 순으로 상위 2개 성향을 결과로 표시
5. 추천 직업, 공부 방식, 점수 막대를 렌더링

## 기술 메모

- 별도 빌드 도구 없이 브라우저에서 바로 실행 가능
- 결과는 `localStorage`의 `career_hint_last_result` 키에 저장
- 키보드 `1`~`4` 입력으로 선택 가능
- 모바일 대응용 반응형 레이아웃 포함

## 현재 분석 요약

- 장점
  - 설치 없이 바로 실행 가능한 구조
  - 질문 수가 짧아 중학생 대상 체험형 콘텐츠에 적합
  - 결과가 단순 점수표를 넘어서 직업/학습 힌트까지 연결됨
- 한계
  - 실행가 성향은 5문항 중 2문항에만 직접 반영되어 상대적으로 노출 빈도가 낮음
  - 분석 결과 설명은 간결하지만 진로군 분류 기준은 더 확장 가능
  - 데이터와 UI가 한 파일에 함께 있어 질문 추가 시 유지보수가 다소 불편함

## 실행 방법

`index.html`을 브라우저에서 열면 바로 실행됩니다.
